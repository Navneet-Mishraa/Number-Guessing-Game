# Number-Guessing-Game

<h2>Overview</h2>
<p>This is a simple yet fun Number Guessing Game where players try to guess a randomly generated number within a specified range. The game provides feedback on whether the guessed number is too high, too low, or correct, making it an engaging and challenging experience for players of all ages.</p>

<h2>How to Play</h2>
<ol>
<li> Start the Game: The game randomly selects a number within a predefined range (e.g., 1 to 100).</li>
<li> Make a Guess: The player inputs their guess.</li>
<li> Get Feedback: The game will tell the player if their guess is too high, too low, or exactly right.</li>
<li> Keep Guessing: Continue making guesses until the correct number is found.</li>
<li> Win the Game: Once the correct number is guessed, the game congratulates the player and optionally displays the number of attempts made.</li>
